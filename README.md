# 10XGrowthEdge – React + Vite + Tailwind

A modern one‑page website for selling Excel templates and business automations.
Includes Hero, Templates, Pricing, FAQ, Contact and a floating WhatsApp button.

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deploy on Vercel
1. Push this folder to GitHub as `10xgrowthedge-site`
2. Go to https://vercel.com → New Project → Import from Git
3. Select the repo, accept defaults (Framework: Vite/React)
4. Deploy → get `*.vercel.app` URL
5. Connect your domain (GoDaddy) in **Settings → Domains** and follow DNS steps.

## Payments
Replace the `href` of the **Buy** buttons with your Instamojo/Gumroad links.
